# 塞壬唱片下载工具

从 [塞壬唱片-MSR](https://monster-siren.hypergryph.com) 下载所有明日方舟的 OST ，包括 wav 文件， lrc 歌词， jpg 封面

## 依赖安装

```bash
pip install -r requirements.txt
```
